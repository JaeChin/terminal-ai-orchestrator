# Verify Implementation

Review all changes made in this session.

## Checks

### Functionality
- Do agent prompts follow the routing rules?
- Are system prompts clear and focused?
- Do scripts run without errors?

### Security
- No API keys or secrets in code?
- `.env.example` updated if new vars added?
- Sensitive data excluded from context files?

### Documentation
- README updated if needed?
- Agent prompts documented?
- Routing logic explained?

### Consistency
- File naming follows conventions?
- Both PS1 and SH versions if scripts modified?
- Context files in sync?

## Output
Report issues by severity:
- **Critical**: Must fix now
- **Warning**: Should fix soon
- **Info**: Nice to have

Fix critical issues before completing.
